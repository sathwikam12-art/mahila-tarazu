# Mahila Tarazu

Mahila Tarazu is a solar powered talking weighing machine designed to help rural women vendors who cannot read numeric displays on traditional weighing machines.

## Problem
Many rural women in Self Help Groups (SHGs) cannot read weight or price on standard weighing machines. This leads to cheating, financial loss, and difficulty in selling products in local markets.

## Solution
Mahila Tarazu is a smart weighing system that:
- Announces weight and price using voice
- Works during night and power cuts using solar power
- Provides LED indicators for correct weighing
- Alerts users in case of cheating or sudden weight manipulation
- Detects disturbances such as theft or animal intrusion

## Features
- Talking weighing system
- Solar powered operation
- Anti-cheating detection
- Visual LED indicators
- Designed for rural women sellers
